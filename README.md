# logic-building-codeN-Cpp

A repository by CodeWithNishchal

A "C++" repository designed to help beginners master logic building before starting DSA.
This repo trains your brain to think logically and helps you improve your logic-building skills step by step.

problem Set: https://drive.google.com/file/d/1qgoctK0fsX9wjxRKJqL90LsOuKM0tBp3/view

- A comment with the question text.
- Beginner-friendly C code for that one question.
- An explanation block describing how the logic works.

Total questions solved: 305

## Folder Structure

- `phase01_conditional_thinking` - Phase 1 - Conditional Thinking (50 files)
- `phase02_looping_patterns` - Phase 2 - Looping And Patterns (65 files)
- `phase03_recursion` - Phase 3 - Recursion (40 files)
- `phase04_basic_arrays` - Phase 4 - Basic Arrays (50 files)
- `phase05_strings` - Phase 5 - Strings (50 files)
- `phase06_mixed_logical_challenges` - Phase 6 - Mixed Logical Challenges (50 files)


## Requirement

Use a C++ compiler to run this project.
